## 5. User Workflows

### Workflow 1: Analyze Purchases by Location
```
1. Navigate to product detail page
2. Click "View Purchase History"
3. Select location from dropdown
4. View filtered purchases and statistics
5. See total units and cost for that location
```

### Workflow 2: Find Which Variants Are at a Location
```
1. Navigate to product detail page
2. Scroll to "Inventory by Location" sidebar
3. View each location card
4. See variant breakdown within each location
```

### Workflow 3: Manage Purchases with Multiple Filters
```
1. Navigate to product edit page
2. Go to "Purchases" tab
3. Select variant from dropdown
4. Select location from dropdown
5. View purchases matching both filters
6. Add/edit/delete purchases as needed
```
